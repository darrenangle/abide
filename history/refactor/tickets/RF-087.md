# RF-087: Build a Verifier-Gated SFT Warmup Dataset Pipeline Across the Current Training Forms

- Priority: `P1`
- Status: `done`
- Area: `training data`
- Origin: `user request`
- Depends On: `RF-084`

## Problem

Abide now has reliable verifiers and a modern Gemma training path, but there
was no first-class synthetic or verifier-gated SFT warmup dataset pipeline for
the current training forms. The repo had prompt generators for RL and scattered
test fixtures for positive poems, but not a clean export path for chat-SFT
records that are guaranteed to pass the active verifier.

## Primary Files

- `src/abide/training/synthetic_sft.py`
- `scripts/build_synthetic_sft_dataset.py`
- `src/abide/training/__init__.py`
- `src/abide_poetry_forms/__init__.py`
- `tests/integration/test_synthetic_sft.py`
- `README.md`
- `data/sft/`

## Acceptance Criteria

- The repo exposes a reusable library function that builds verifier-gated SFT
  records for at least the current training surface.
- The repo includes a CLI/export script that writes those records to JSONL.
- The exported records include explicit `form_name` metadata and chat-style
  prompt/response structure suitable for SFT.
- Focused tests cover form coverage, verifier gating, and the export path.

## Outcome

- Added `abide.training.synthetic_sft` with a verifier-gated seed corpus and
  reusable helpers to build, summarize, and write chat-SFT records.
- Added `scripts/build_synthetic_sft_dataset.py` for JSONL export.
- Exported the builder through both `abide.training` and `abide_poetry_forms`.
- Added integration coverage for full `rl_default` coverage, missing synthetic
  coverage failure, and CLI export behavior.
- Wrote a concrete artifact at `data/sft/abide_synthetic_rl_default.jsonl` and
  `data/sft/abide_synthetic_rl_default.summary.json`.

## Notes

- The current `rl_default` artifact mixes `synthetic` and `public_domain`
  positive seeds so the entire training subset is covered without pretending the
  public-domain sonnet and rondeau fixtures are synthetic.
- All `rl_default` seeds pass the active training-profile verifier. Two of the
  public-domain seeds score below `1.0` but still pass canonically:
  `ShakespeareanSonnet` (`0.9595`) and `Rondeau` (`0.9688`).
- Exported artifact summary: `112` records total, `14` forms, `8` prompt
  variants per seed, `88` synthetic rows, `24` public-domain rows.
