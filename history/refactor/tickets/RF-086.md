## Problem

The repo has a validated modern Gemma 4 E2B `prime-rl` path and a durable
`well_known` baseline, but the user now wants a larger real run that expands
to the broader `rl_default` subset: the original well-known forms plus the
lesser-known rollout forms (`Triolet`, `Pantoum`, `TerzaRima`, `Rondeau`,
`Clerihew`, and `BluesPoem`).

The existing runner profiles are biased toward the validated `well_known`
recipes. This task is to stage and launch a larger mixed-form job with heavier
rollout counts, record the exact runtime recipe, and capture the resulting
artifact and telemetry paths.

## Acceptance Criteria

- A tracked large-run recipe is defined for the modern Gemma 4 E2B path using
  `form_set=rl_default`.
- The launched run uses W&B online logging rather than local offline-only mode.
- The run uses a heavier rollout budget than the existing `mixed-stable`
  baseline and records the exact launch parameters in the tracker.
- The ticket records the output directory and any directly observed early
  metrics or blockers.
