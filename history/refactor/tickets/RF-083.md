# RF-083: Run Real Gemma 4 E2B Mixed-Form Training Sweeps

- Priority: `P1`
- Status: `done`
- Area: `training reliability`
- Origin: `PRIME-RL-MIXED-SWEEP-2026-04-07`

## Problem

The modern Gemma 4 E2B `prime-rl` path is now structurally sound and has
verified short-form and long-form canaries, but we still need an actual
mixed-form training pass with parameter sweeps instead of more plumbing work.
The current repo does not have a first-class sweep harness for the modern
`prime-rl` path, and we do not yet have a directly verified mixed-form run that
looks good enough to use as the next real training baseline.

## Primary Files

- `scripts/`
- `tests/integration/`
- `history/refactor/`

## Acceptance Criteria

- The repo must expose a repeatable sweep path for the modern Gemma 4 E2B
  `prime-rl` runner.
- At least one mixed-form Gemma 4 E2B run must complete cleanly with credible
  reward metrics and no runner/runtime failures.
- The sweep results and chosen baseline must be recorded in the tracker.
