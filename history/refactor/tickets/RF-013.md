# RF-013: Add Inference Regression Coverage And Narrow Guarantees

- Priority: `P1`
- Status: `in_progress`
- Area: `form inference`
- Origin: `FOLLOWUP-005`

## Problem

The inference module exists and can often produce useful `FormSpec`s, but the repository does not currently carry direct test coverage proving the strong README-style guarantee that inferred specs pass source poems at `1.0` across representative examples.

## Primary Files

- `src/abide/inference/analyzer.py`
- `tests/`
- `README.md`

## Acceptance Criteria

- Add direct tests for `analyze_poem()` and `infer_form()` on representative fixture poems.
- Confirm which guarantees hold consistently, and narrow doc claims where they do not.
- Keep the verified guarantee explicit and reproducible.
