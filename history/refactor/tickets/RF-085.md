# RF-085: Use RF-084 Holdout Artifacts to Improve Mixed-Stable Generalization on Weak Well-Known Forms

- Priority: `P1`
- Status: `pending`
- Area: `training reliability`
- Origin: `RF-084`

## Problem

`RF-084` proved that the modern Gemma 4 E2B `mixed-stable` path trains cleanly
across multiple seeds, but it is not yet good enough on deterministic holdout
evaluation. The recorded two-seed pass landed at mean training reward `0.9417`
but only mean holdout pass rate `0.375`.

The failures are not random:

- `Sestina` fails in both seeds by breaking the stanza-two end-word cycle and,
  in one case, the envoy line count.
- `Tanka` fails in both seeds on exact syllable counts.
- `Ghazal` fails in both seeds by missing radif / matla structure.
- `Villanelle` is unstable across seeds, with one seed passing cleanly and the
  other missing refrain placement / stanza size on one sample.
- `PetrarchanSonnet` scores high but still misses strict rhyme closure on
  holdout prompts.

The next work is to convert those artifacts into a better training recipe
instead of relying on in-run reward alone.

## Acceptance Criteria

- The repo must define a follow-up training recipe or curriculum that directly
  targets the weak well-known forms identified by `RF-084`.
- At least one revised validation pass must be run against the same
  deterministic holdout seed set used by `RF-084`.
- The revised pass must materially improve the mean holdout pass rate over the
  `0.375` RF-084 baseline, with explicit per-form metrics recorded in the
  tracker.

## Suggested Direction

- Split the current mixed recipe into a curriculum or weighted schedule that
  spends more budget on `Sestina`, `Tanka`, `Ghazal`, `Villanelle`, and
  `PetrarchanSonnet`.
- Consider per-form prompt/budget tuning instead of a single mixed budget.
- Keep the RF-084 holdout prompt set and export path fixed so comparisons stay
  honest.
